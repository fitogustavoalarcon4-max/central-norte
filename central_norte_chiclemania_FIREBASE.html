📊 Resumen del día
🧪 Modo Prueba Solo Admin
Cargá stock de prueba para verificar que el sistema funcione correctamente. Podés borrar todo después sin afectar el catálogo.
Paso 1 — Cargar stock de prueba
Paso 2 — Probá como vendedor
Cerrá sesión, entrá como Vendedor, hacé una venta de prueba y verificá que el stock baje correctamente.
Paso 3 — Limpiar datos de prueba
Esto borra todas las ventas y movimientos de prueba y deja el stock en 0. El catálogo de productos NO se toca.
🔧 Recalcular stock 🔐 Admin
Si el stock quedó incorrecto por anulaciones múltiples u otros errores, usá esto para recalcularlo desde cero — sumando todos los ingresos de bodega y restando solo las ventas activas (las anuladas no se cuentan). El catálogo y el historial no se tocan.
🔴 Reinicio total del sistema 🔐 Admin
Usá esto cuando estés listo para arrancar el primer mes real con la app. Borra absolutamente todo el historial acumulado hasta hoy — ventas, clientes, stock, caja, pedidos bodega, pedidos sugeridos, auditorías y movimientos — para que empieces desde cero con datos limpios. El catálogo de productos NO se borra, solo las cantidades vuelven a 0 para que cargues el inventario real.
⚠️ Esta acción NO se puede deshacer. Asegurate de haber exportado a Excel cualquier dato de prueba que quieras conservar antes de continuar.
